== Installation ==

In English :

1.  Upload the youtube-devfacile folder to the /wp-content/plugins/ directory
2.  Activate the YouTube Dev Facile plugin through the 'Plugins' menu in WordPress
3.  Configure the plugin by going to the YouTube Dev Facile menu that appears in your admin menu
4.  Paste shortcode anywhere including any sidebar text widget : [YTcount select="nbVideos"] or [YTcount select="nbViews"]
or use this php code : echo do_shortcode('[YTcount select="nbVideos"]'); or echo do_shortcode('[YTcount select="nbViews"]');

En Français :

1.  Extraire le fichier zip et déposez simplement le contenu dans le répertoire wp-content/plugins/ de votre installation de WordPress.
2.  Activez le plugin depuis la page Plugins.
3.  Configurez le plugin depuis le menu YouTube Dev Facile dans le menu Réglages de Wordpress.
4.  Copiez ce shortcode n'importe où, dans un article, dans un widget... : [YTcount select="nbVideos"] ou [YTcount select="nbViews"]
ou utilisez ce code php : echo do_shortcode('[YTcount select="nbVideos"]'); ou echo do_shortcode('[YTcount select="nbViews"]');
